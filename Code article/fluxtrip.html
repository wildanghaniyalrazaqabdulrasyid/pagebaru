FluxTrip Mobile App






Introduction

The Fluxtrip travel app has always been my favorite to use. Fluxtrip is one of the largest travel and vacation accommodation providers that can be used as a reference for many people. This company is known as a company that relies on data when building and improving their products.

This concept is purely aimed at improving their visual application so that it will be more attractive for users to use the product. In this case, I also tried to provide a new flow for the login flow and the main menu.

PROCESS Starting to try and learn how the Fluxtrip application works is the first part of the double diamond process which tries to determine the problems that I need to solve. I started to notice some things that I thought could be improved.

The main focus of this redesign exercise project is:
- improving the user flow for login.
- improving the visual style
